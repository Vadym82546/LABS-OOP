Гра "Морський бій".
За допомогою графічного інтерфейсу гравець має можливість обрати розташування кораблів (з відображенням коректності розташування) та подалі робити постріли.
Гра відбувається за класичними правилами.
Для компьютера імплементовано простий алгоритм пострілів.

Вихідні дані: наявні скріншоти різних етапів гри та приклади (не)можливого розташування кораблів.
