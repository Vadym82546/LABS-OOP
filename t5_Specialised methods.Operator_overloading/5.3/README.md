Скрипт шукає інпут-файл та обчислює арифметичні вирази записані у файлі.
Реалізовано конструктор з можливістю копіювання, що може приймати
• два цілих аргументи – відповідно чисельник та знаменник дробу;
• один аргумент (рядкового типу) у якому записується раціональне число у вигляді дробу 'n/d'
Також створено методи для отримання чисельника та знаменника дробу та можливість отримати значення дробу у вигляді десяткового числа.
